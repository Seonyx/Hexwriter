# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Hexwriter is a standalone book-editor web application at Hexwriter.com. It is being extracted from the Seonyx.com codebase and extended with a multiuser permissions system.

## Stack

- **Backend**: ASP.NET (.NET Framework 4.5), C#
- **Development workflow**: Code is written on Linux (this machine) with Claude Code / VSCode, then copied to Windows 10 and built/tested in Visual Studio. There is no Linux build step — do not attempt to compile or run the project locally.

## Repository State

The codebase is being populated by lifting code from the Seonyx.com book-editor. Until that extraction is complete this file will be updated to reflect the actual project structure.
